# AnalyzePitch Frontend

A modern React-based frontend application for AI-powered pitch analysis and evaluation.

## Features

- **Modern React UI**: Built with React and modern JavaScript
- **Pitch Upload**: Upload and analyze pitch documents
- **Real-time Analysis**: Get instant AI-powered feedback
- **Interactive Dashboard**: View analysis results and insights
- **Responsive Design**: Works seamlessly on desktop and mobile

## Prerequisites

- Node.js 16+ and npm (or yarn)
- Backend API running (see backend README)

## Installation

### 1. Navigate to Frontend Directory

```bash
cd AnalyzePitch/frontend
```

### 2. Install Dependencies

**Using npm:**
```bash
npm install
```

**Using yarn:**
```bash
yarn install
```



# Backend API URL
REACT_APP_API_URL=http://localhost:8000


## Running the Application

### Development Server

**Using npm:**
```bash
npm run dev
```

**Using yarn:**
```bash
yarn start
```

The application will open at `http://localhost:8080`
